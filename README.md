# Project Overview

This project simulates a real-world pricing analytics backend used by businesses to understand how their products perform across different countries and markets.
The backend collects pricing and sales data, processes it using analytics logic, and exposes meaningful insights through RESTful APIs. These insights can be consumed by dashboards, reporting tools, or other services.
